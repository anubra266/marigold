Icon

With the Icon component you can render one of the Marigold Icons. By adding the size prop you can change to another size. 24 is the default size. The fill prop is the second attribute and per default currentcolor.

React Icon component

Marigold Icons have a separate package to install. The first option is to install the package and use the Icons like other React components and as in the code example below. You have to specify the name of the icon directly between the < /> brackets.

# with npm
npm install @marigold/icons --save


# with yarn
yarn add @marigold/icons


import { DesignTicket } from '@marigold/icons';


HTML SVG Element

The second option is to go to Marigold Icons and click on an Icon in the list to copy them as <svg> Element
