NumberField

Description

The NumberField components allows users to enter numbers with a keyboard. This includes plain numbers as well as percentages, decimals and sums of money. It consists of a label, an input field and a help text. The later also includes error messages.

You should really try this!
import { NumberField } from '@marigold/components';

Props

PropertyTypeDefaultDescription
labelReactNodenoneThe label text.
descriptionReactNodenone A helpful text.
errorMessageReactNodenoneAn error message.
errorbooleanfalseIf `true`, the field is considered invalid and if set the errorMessage is shown instead of the `description`.
valuenumbernoneThe value of the input field.
disabledbooleanfalseIf `true`, the input is disabled.
requiredbooleanfalseIf `true`, the input is required.
readOnlybooleanfalseIf `true`, the input is readOnly.
onChangefunctionA callback function that is called with the input's current value when the input value changes.
formatOptionsIntl.NumberFormatOptionsApply formatting. See `Intl.NumberFormatOptions` for available options.
widthstring100%Control the width of the field.
...Yes you can use all regular attributes of input!

Examples

Simple Number Field

import { NumberField } from '@marigold/components';


export const BasicNumberField = () => <NumberField label="Numbers" />;

Disabled Number Field

import { NumberField } from '@marigold/components';


export const DisabledNumberField = () => (
  <NumberField label="Field disabled" disabled />
);

Required Number Field

import { NumberField } from '@marigold/components';


export const RequiredNumberField = () => (
  <NumberField label="Numbers" required />
);

Number Field with an Error

Something went wrong!
import { NumberField } from '@marigold/components';


export const ErrorNumberField = () => (
  <NumberField label="Amount" error errorMessage="Something went wrong!" />
);

Min and max Values

import { NumberField } from '@marigold/components';


export const ValuesNumberField = () => (
  <NumberField label="Length" defaultValue={55} minValue={0} maxValue={100} />
);

Step Amount

import { NumberField } from '@marigold/components';


export const AmountNumberField = () => (
  <NumberField label="Length" defaultValue={10} step={10} />
);

Hide Stepper

import { NumberField } from '@marigold/components';


export const HideNumberField = () => (
  <NumberField label="Numbers" hideStepper />
);

Diplay a Decimals

import { NumberField } from '@marigold/components';


export const DecimalNumberField = () => (
  <NumberField
    label="Decimals"
    formatOptions={{
      signDisplay: 'exceptZero',
      minimumFractionDigits: 1,
      maximumFractionDigits: 2,
    }}
    defaultValue={0}
  />
);

Diplay a Currency

import { NumberField } from '@marigold/components';


export const CurrencyNumberField = () => (
  <NumberField
    label="Amount"
    defaultValue={19.99}
    formatOptions={{
      style: 'currency',
      currency: 'USD',
    }}
  />
);

Diplay a Percentage

import { NumberField } from '@marigold/components';


export const PercentageNumberField = () => (
  <NumberField
    label="Percentage"
    defaultValue={0.42}
    formatOptions={{
      style: 'percent',
    }}
  />
);

Display a Unit

import { NumberField } from '@marigold/components';


export const UnitNumberField = () => (
  <NumberField
    label="Length"
    defaultValue={150}
    minValue={0}
    formatOptions={{
      style: 'unit',
      unit: 'centimeter',
      unitDisplay: 'short',
    }}
  />
);
