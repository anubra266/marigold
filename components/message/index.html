Message

Description

Simple component to display a message with a title and icon based on a div element and styled with a theme.

Properties

PropertyTypeDefault
messageTitlestring
variant (optional)stringinfo

Import

import { Message } from '@marigold/components';


Live-Code Example

Warning!

This is a warning.
<Message variant="warning" messageTitle="Warning!">
  <Text>This is a warning.</Text>
</Message>
