Columns

Description

With the Columns component you can add a bundle of column components by adding props to create a layout.

Properties

PropertyTypeDefault
space (optional)number0
horizontalAlign (optional)left, right, centerleft
verticalAlign (optional)top, bottom, centertop

Import

import { Columns } from '@marigold/components';


Live-Code Example

Here's your text
Here's antoher text
<Columns verticalAlign="top" horizontalAlign="left">
  <Column width={6}>
    <Text>Here's your text</Text>
  </Column>
  <Column width={6}>
    <Text>Here's antoher text</Text>
  </Column>
</Columns>
