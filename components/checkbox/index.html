Checkbox

Description

With the Checkbox component you can add a HTML <input> element with type="checkbox" to your form. The variant can be added with the variant prop. The default variant is default.

Properties

Property
Type
Default

id

string

variant (optional)

string

default

label (optional)

string

required (optional)

boolean

Import

import { Checkbox } from '@marigold/components';


Live-Code Example

() => {
  const [isChecked, setChecked] = React.useState(false);
  const onChange = () => {
    setChecked(!isChecked);
  };
  return (
    <Checkbox
      id="checkboxLive"
      name="checkboxLive"
      onChange={onChange}
      checked={isChecked}
      value="Checkbox"
      label="Checkbox to play with"
    />
  );
};
