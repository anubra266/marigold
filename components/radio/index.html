Radio

View in Figma

The Radio and Radio.Group components allow users to select one option from a set of options. They are useful for presenting a list of options to users and collecting their responses.

Choose desert
import { Radio } from '@marigold/components';

Props

PropertyTypeDefaultDescription
variantstringnoneUse a different _variant_ from theme
sizestringnoneUse a different _size_ from theme
errorbooleanfalseIf `true`, the checkbox is considered invalid and if set the `errorMessage` is shown instead of the `description`.
widthstring100%Control the width of the field

Examples

Simple Radio

Radio Group
import { Radio } from '@marigold/components';


export const SimpleRadio = () => (
  <Radio.Group label="Radio Group">
    <Radio value="1">Option 1</Radio>
    <Radio value="2">Option 2</Radio>
    <Radio value="3" disabled>
      Option 3
    </Radio>
    <Radio value="4">Option 4</Radio>
  </Radio.Group>
);
