Radio Button

Description

With the Radio Button component you can add a HTML <input> element with type="radio" to your form. The variant can be added with the variant prop. The default variant is radio.

Properties

PropertyTypeDefault
idstring
variant (optional)stringradio
label (optional)string
required (optional)boolean

Import

import { Radio } from '@marigold/components';


Live-Code Example

<Radio checked id="live" value="live" label="Radio label" />
