Center

Center is a layout component that centers its children horizontally. You can use it to center a single child or a list of children.

import { Box, Center } from '@marigold/components';


export const BasicCenter = () => (
  <Center maxWidth="min(350px, 50vw)">
    <Box border="1px solid #ced4da" bg="#e9ecef" height="100px" width="100%" />
  </Center>
);

Props

PropertyTypeDefaultDescription
spaceResponsiveStyleValue<string>noneThe space between the children
maxWidthstringThe maximum width of the container.

Examples

Center element with maximum width 

import { Box, Center } from '@marigold/components';


export const WidthCenter = () => (
  <Center maxWidth="xxlarge">
    <Box border="1px solid #ced4da" bg="#e9ecef" height="100px" width="100%" />
  </Center>
);

Center more than one element with space 

import { Box, Center } from '@marigold/components';


export const SpaceCenter = () => (
  <Center maxWidth="xxlarge" space="xsmall">
    <Box border="1px solid #ced4da" bg="#e9ecef" height="100px" width="100%" />
    <Box border="1px solid #ced4da" bg="#e9ecef" height="100px" width="100%" />
    <Box border="1px solid #ced4da" bg="#e9ecef" height="100px" width="100%" />
  </Center>
);

Center something inside a box

Song 1

import { Box, Center, Inline, Text } from '@marigold/components';
import { Stop, Play, Pause } from '@marigold/icons';


export const BoxCenter = () => (
  <Inline space="xsmall">
    <Box as={Center} border="2px solid #ced4da" width="80px" height="40px">
      <Text color="secondary">Song 1</Text>
    </Box>
    <Box as={Center} bg="secondary" width="40px" height="40px">
      <Play fill="white" />
    </Box>
    <Box as={Center} bg="secondary" width="40px" height="40px">
      <Pause fill="white" />
    </Box>
    <Box as={Center} bg="secondary" width="40px" height="40px">
      <Stop fill="white" />
    </Box>
  </Inline>
);
