Box

The all-rounder among the components.

The <Box> is a basic component like the HTML div element. You can modify the component with different style props. It is also possible to write __baseCSS and css styles on the <Box>.

You can use the as prop to let the <Box> behave as any Marigold component or HTML Element with all its properties.

For a more detailed description, see Box Primitive .

I am basic div box!

Usage

Import

To import the component you just have to use this code below.

import { Box } from '@marigold/components';

Examples

Basic Box

In this example the <Box> is modified with several props.

Box Example

import { Box, Text } from '@marigold/components';


export const SuccessBox = () => (
  <Box bg="success" p="8px" width="25%" display="flex" justifyContent="center">
    <Text color="white">Box Example</Text>
  </Box>
);

Box as Button

The as prop is set to a button HTMLElement on the left side and to a Marigold button on the right.

import { Box, Button, Inline } from '@marigold/components';


export const ButtonBox = () => (
  <Inline space="medium">
    <Box as="button" bg="primary" p="8px">
      I am a button
    </Box>
    <Box as={Button} variant="primary">
      I am a button
    </Box>
  </Inline>
);

Box with several props

How to use different props and CSS on the <Box>.

I am a paragraph with a lot of props!

import { Box } from '@marigold/components';


export const TextBox = () => (
  <Box
    as="p"
    border="1px solid"
    borderRadius="5px"
    p="8px"
    css={{ bg: 'lightseagreen', color: 'white' }}
  >
    I am a paragraph with a lot of props!
  </Box>
);
