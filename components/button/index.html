Button

Description

With the Button component you can render a button element. The style variant and size of the button can be added with the variant prop and is composed like variant.size.

Properties

PropertyTypeDefault
variant (optional)primary.small, primary.large, secondary.small, secondary.large, ghost.small, ghost.largeprimary.large

Import

import { Button } from '@marigold/components';


Live-Code Example

<Button
  variant="secondary.small"
  onClick={() => {
    alert('Searching..');
  }}
>
  Search on <Google /> oogle
</Button>
