Slider

Description

With the Slider component you can add a HTML <input> element with type="range" to your form. The variant can be added with the variant prop. The default variant is slider.

Properties

Property
Type
Default

variant (optional)

string

slider

Import

import { Slider } from '@marigold/components';


Live-Code Example

<Slider id="vol" name="vol" min="0" max="50" />
